# EDA of Public Service Expenditure Disparities

## Overview

This project focuses on performing Exploratory Data Analysis (EDA) on public service expenditure data to identify disparities between Hispanic and White non-Hispanic consumer groups across multiple demographic categories. The analysis was conducted using Python to clean, process, visualize, and interpret real-world data in order to uncover meaningful patterns and insights.

The goal of this project was to strengthen analytical thinking and demonstrate the ability to work with datasets through data cleaning, transformation, statistical exploration, and visualization techniques.

## Objectives

* Clean and prepare raw datasets for analysis
* Explore trends and spending disparities across demographic groups
* Identify patterns and potential correlations within the data
* Visualize findings using clear and informative charts
* Generate actionable insights from real-world public expenditure data
